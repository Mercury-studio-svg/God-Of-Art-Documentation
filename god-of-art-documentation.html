Introduction

Overview

God of Art is a cinematic, game-ready ancient temple environment pack built in Unreal Engine. Inspired by Khmer and Hindu architectural traditions, it delivers AAA-quality visuals — hand-sculpted stone idols, crumbling ruins, jungle overgrowth, and dramatic torch-lit atmosphere.

This is a Blueprint-only project. No C++ knowledge is required. The pack includes a fully built demo level with cinematic lighting, Lumen Global Illumination, particle effects, and all assets organized for easy migration into your own project.

Lumen GI Ready
Full dynamic global illumination with warm torch-lit volumetric atmosphere.
Nanite Compatible
High-poly meshes optimized with Nanite for maximum visual fidelity.
4K PBR Textures
Full PBR pipeline — Base Color, Normal, Roughness, Metallic up to 4096×4096.
Demo Level Included
A pre-built cinematic scene ready to render or use as a starting point.

Setup

Installation

1
Purchase & Download from Fab
Go to fab.com, find God of Art by Campass Studios, and click Add to My Library. Then open Epic Games Launcher → Library → Fab Library.
2
Add to Project
Click Add to Project next to God of Art and select your Unreal Engine project. The assets will be added to your Content folder automatically.
3
Open Unreal Engine
Launch your project. In the Content Browser, navigate to Content → GodOfArt to find all assets.
4
Open the Demo Level
Navigate to Content → GodOfArt → Maps → DemoLevel and double-click to open the pre-built cinematic scene.
Recommended Engine Version

For best results with Lumen and Nanite features, use Unreal Engine 5.0 or above. The pack is also compatible with UE 4.20, 4.22, and 4.26 (Lumen/Nanite not available in UE4).

Quick Start

Migrate to Your Project

To move assets from the demo project into your own game project:

1
Select Assets
In the Content Browser, select the GodOfArt folder or individual assets you want.
2
Right Click → Migrate
Right-click the selected assets → Asset Actions → Migrate.
3
Choose Destination
Navigate to your target project's Content folder and click Select Folder. All dependencies are copied automatically.

Usage

Using the Demo Level

The included demo level is a fully lit, camera-ready scene. You can use it as-is for renders, or as a reference to build your own scenes.

To Render a Cinematic Screenshot

Open DemoLevel → Press G to hide editor UI → Press Ctrl + Shift + F11 to enter high-res screenshot mode → Set multiplier to 4x for ultra quality.

Customization

Customizing Materials

All materials use a master material + instance workflow. To change colors, roughness, or texture tiling on any asset:

1
Find the Material Instance
In Content Browser → GodOfArt/Materials/Instances, find the instance used by your asset.
2
Double-click to Edit
Open the instance and enable the checkboxes next to parameters you want to override — color, roughness, normal intensity, and tiling.
3
Save
Press Ctrl+S to save. Changes apply instantly in the viewport.

Optimization

Performance Tips

For use in real-time games (vs cinematic renders), consider these optimizations:

✓
Enable Nanite on Heavy Meshes
Select the stone idol meshes → Details panel → Enable Nanite for automatic polygon streaming (UE5 only).
✓
Use Cull Distance Volumes
Add a Cull Distance Volume around dense foliage areas to automatically hide small meshes at distance.
✓
Lower Texture Streaming Pool
If you see texture pop-in, increase the texture streaming pool in Project Settings → Rendering.

Help

Frequently Asked Questions

Does this work in Unreal Engine 4?

Yes — compatible with UE 4.20, 4.22, and 4.26. However Lumen and Nanite features are only available in UE5. In UE4, static lighting (baked) will be used instead.

Do I need C++ to use this?

No. This is a 100% Blueprint project. No programming knowledge is required.

Can I use this in a commercial game?

Yes. A standard Fab license allows use in commercial games and applications. See Fab's licensing terms for full details.

Can I modify the assets?

Yes. All materials and meshes are fully editable. You can adjust, retexture, or resize any asset to fit your project.
